## 🔧 Configuration

### Default Settings
- Margin: 1% below lowest competitor
- Report frequency: Every 10 agent cycles
- Alert threshold: 5% above lowest

### Easy to Customize
```python
# In your code
engine = PriceIntelligenceEngine(db, margin_percent=2.0)  # 2% instead of 1%

# In agent_engine.py
self._pricing_report_interval = 5  # Every 5 cycles instead of 10

# For alerts
alerts = await engine.create_price_alerts(threshold_percent=10.0)
```
